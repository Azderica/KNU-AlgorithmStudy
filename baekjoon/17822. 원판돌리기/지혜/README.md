# 🌀원판돌리기🌀

https://www.acmicpc.net/problem/17822



## 풀이과정

1. 원판 돌리기 

2. 인접해 있는데 같은 값 수 지우기

3. 숫자 지우기 안 했으면 원판에 남은 숫자 평균 구해서 작은애는 +1 큰애는 -1

   숫자 지우기 했으면 원판에 남은 숫자 합 구하기
